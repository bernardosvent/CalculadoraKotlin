# Calculadora Simples - Jetpack Compose 📱

Projeto de desenvolvimento de uma calculadora simples utilizando Android Studio, Kotlin e Jetpack Compose. Este aplicativo foi construído como parte das atividades práticas do curso de Engenharia de Software.

## 🎓 Contexto Acadêmico

* **Instituição:** CENTRO UNIVERSITÁRIO SATC (UNISATC)
* **Curso:** Engenharia de Software
* **Disciplina:** Soluções Mobile
* **Professor:** Thyerri Mezzari

## 🎯 Objetivo do Projeto

O objetivo principal deste exercício é desenvolver uma calculadora simples para fixar os conceitos do Jetpack Compose, iniciando pela camada visual para montagem da tela e, em seguida, partindo para a parte lógica.

## ✨ Funcionalidades

* Interface construída com Jetpack Compose.
* Display numérico utilizando o componente `Text`.
* Operações matemáticas básicas:
  * Soma (+)
  * Subtração (-)
  * Multiplicação (*)
  * Divisão (/)
* Botão "C" para limpar o visor e resetar o estado da calculadora.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Kotlin
* **IDE:** Android Studio
* **UI Toolkit:** Jetpack Compose

1. Clone este repositório: `git clone https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git`

## 🎯 Objetivo do Projeto

[cite_start]O objetivo principal deste exercício é fixar os conceitos visuais e lógicos do Jetpack Compose[cite: 24]. [cite_start]A interface foi construída primeiramente na camada visual para o entendimento do layout [cite: 24][cite_start], seguida pela implementação da lógica de funcionamento matemático[cite: 25].

## ✨ Funcionalidades

* [cite_start]Interface moderna baseada nos componentes do Material Design 3[cite: 28].
* [cite_start]Display numérico alinhado à direita utilizando o componente `Text`[cite: 26].
* Operações matemáticas básicas ativas:
  * [cite_start]Soma (+) [cite: 8, 22]
  * [cite_start]Subtração (-) [cite: 8]
  * [cite_start]Multiplicação (*) [cite: 8, 19]
  * [cite_start]Divisão (/) [cite: 8, 13]
* [cite_start]Botão de limpar (C) para resetar o estado da calculadora[cite: 21].
* [cite_start]Layout responsivo que preenche todo o espaço da tela de forma harmoniosa[cite: 27].
